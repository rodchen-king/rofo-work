<h1 align="center">rofo-work</h1>

An project to help front software development.

## Features

- :主页: 功能列表
- :前端须知: 项目开发规范，新人培训，项目介绍
- :项目组件: 公司内部组件沉淀
- :知识分享: 项目分享计划，实施
- :工作记录: 工作相关内容【待思考】

## Usage

### Use bash

```bash
$ npm install
$ npm start         # visit http://localhost:3000
```
